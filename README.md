# Telegram bot for the DevOps course from Prometheus



## Requirement
 - Go(https://go.dev/dl/)
 - Telegram Token(Bot Father)
 - Dep: github.com/spf13/cobra та gopkg.in/telebot.v3


 ## Build
 ```bash
 go get github.com/spf13/cobra 
 go get gopkg.in/telebot.v3
 go build
 ```

 ## Run
 ```bash
 source TOKEN <value>
 ./main start_bot
 ```
